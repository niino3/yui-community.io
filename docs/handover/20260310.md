## 3. 設計方針メモ

- **認証済み → 実API** / **未認証 → モックデータ** にフォールバック
- mutation 失敗時もモックフローで UI は進行（ユーザー体験を壊さない）
- React Query で自動キャッシュ・再フェッチ
- `VITE_NETWORK=mainnet` で本番、省略時は Amoy テストネット
