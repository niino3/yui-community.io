## 3. 3サイト構成アーキテクチャ

### URL 設計（ローカル開発）

| サイト | アクター | URL | レイアウト |
|--------|---------|-----|-----------|
| Site 1 | Service Owner | `http://localhost:5173/` | デスクトップ フルページ |
| Site 2 | Community Owner | `/?community={slug}&admin=true` | デスクトップ フルページ |
| Site 3 | Community Member | `/?community={slug}` | モバイルフレーム |

### 判定ロジック（`App.jsx`）

```
slug なし        → LandingPage（Site 1）
slug + admin=true → TenantAdmin フルページ（Site 2）
slug のみ        → モバイルアプリ（Site 3）
```

### URL 設計（本番予定）

| サイト | URL |
|--------|-----|
| Site 1 | `yui-community.io` |
| Site 2 | `{slug}.yui-community.io?admin=true` |
| Site 3 | `{slug}.yui-community.io` |

## 6. ローカル開発環境

### 起動手順

```bash
# バックエンド（Docker）
cd backend
docker-compose up

# フロントエンド（別ターミナル）
cd yui-community.io
npm run dev
```

### 環境変数（.env）

```
VITE_API_URL=http://localhost:8080
```

### Docker コンテナ

| コンテナ | ポート | 用途 |
|---------|--------|------|
| yui-app | — | PHP-FPM（Laravel） |
| yui-nginx | 8080 | Nginx リバースプロキシ |
| yui-db | 5432 | PostgreSQL |
| yui-redis | 6379 | Redis |

### よく使うコマンド

```bash
docker-compose exec app php artisan migrate          # マイグレーション
docker-compose exec app php artisan config:clear     # 設定キャッシュクリア
docker-compose exec app php artisan route:list       # ルート一覧
```
