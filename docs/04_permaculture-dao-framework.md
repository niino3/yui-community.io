## 思想的背景：なぜパーマカルチャーとDAOは親和性が高いのか

パーマカルチャー（持続可能な永続的農業・文化）とDAO（自律分散型組織）は、共通のゴールを持っている。

> **「中央集権的な管理に頼らず、個々が自律しながら全体の生態系を豊かにする」**

この視点を本プロジェクトの設計思想の核に置く。

---

## 1. パーマカルチャーの3倫理 × DAO実装

パーマカルチャーの3つの基本倫理をDAOの文脈で整理すると、地域通貨の役割がより明確になる。

| パーマカルチャーの倫理 | DAOにおける実装・解釈 | 地域通貨（Web3）での役割 |
|----------------------|---------------------|------------------------|
| **地球への配慮**（Earth Care） | 環境再生に貢献する活動の可視化 | 堆肥作り・植樹など環境負荷を下げる行動に対してトークンを発行 |
| **人への配慮**（People Care） | 相互扶助の仕組みをスマートコントラクトで構築 | 介護・育児・農作業の「お手伝い」を記録し、信頼スコアとして蓄積 |
| **余剰の分かち合い**（Fair Share） | 利益や資源を独占せず、コミュニティに再投資 | 余った作物をトークンで循環させ、余剰トークンを共有財の整備に充てる |

## 2. 地域通貨を「生態系（エコシステム）」として捉える

パーマカルチャーには **「多機能性（ひとつの要素が複数の役割を持つ）」** という原則がある。これをWeb3の地域通貨に当てはめると、単なる「お金」以上の機能を持たせることができる。

### 2.1 通貨が「土壌（データ）」になる

ブロックチェーン上の取引履歴は、その土地で **誰が・いつ・何を作ったか** という「活動の足跡」。

これが蓄積されることで：
- 地域のニーズ（今、誰が何を必要としているか）のリアルタイム可視化
- 最適な資源配分の実現
- 外部の研究者・支援者への活動実績の提示

が可能になる。

### 2.2 貢献が「栄養（トークン）」になる

「草刈りをした」「種を分けた」といった行動がトークンとして還元される。  
これは **植物が枯れて土に還り、次の命の栄養になるサイクル** と同じ。

```
貢献行動
  ├─ People Care（お手伝い・農産物提供）
  │      ↓ Proof of Help
  └─ Earth Care（堆肥・植樹・農薬不使用）
         ↓ Proof of Earth Care
         ↓
  （上位概念：Proof of Contribution）
  = 労働・貢献をデジタル証明書（SBT）として記録
         ↓
  トークン発行（栄養として循環）
         ↓
  別の誰かの貢献行動を引き出す
```

> **用語体系：**  
> `Proof of Contribution`（上位概念）= `Proof of Help`（People Care）+ `Proof of Earth Care`（Earth Care）  
> 実装面では各サブタイプのSBTを発行し、ユーティリティトークンを付与する。詳細は [02_business-requirements.md §2.2](./02_business-requirements.md) 参照。

### 2.3 ガバナンスが「剪定（調整）」になる

DAOの投票機能を使って、コミュニティの方向性を定期的に「剪定」する。

**投票例：**
- 今年の共同農園にどの種を植えるか？
- 余ったトークンをどのお祭りの費用に使うか？
- 農機具の共同購入優先順位は？

これは、健全な成長のために不要な枝を切り、光を当てる「剪定」の作業に対応する。

## 3. トークンの3レイヤー設計（種・花・循環）

Web3技術を使って実装する際、以下の3つのレイヤーで考えるとスムーズ。

```
┌──────────────────────────────────────────┐
│  Layer 3: 価値の循環（循環）              │
│  使われなかったトークンが自動的に          │
│  地域の共用道具（耕運機等）の             │
│  メンテナンス費用に回る                   │
│  → Demurrage / 自動Treasury送金          │
└──────────────┬───────────────────────────┘
               │ 余剰トークンが流れ込む
┌──────────────▼───────────────────────────┐
│  Layer 2: 価値の交換（花）               │
│  「お手伝い」と「作物」を                 │
│  スムーズに交換するアプリ（UI）           │
│  → マッチングプラットフォーム            │
└──────────────┬───────────────────────────┘
               │ 信頼が積み重なる
┌──────────────▼───────────────────────────┐
│  Layer 1: 価値の保存（種）               │
│  コミュニティ内での信頼を                 │
│  トークンとしてストック                   │
│  → SBT・信頼スコア・残高                 │
└──────────────────────────────────────────┘
```

## 4. 「多機能性」原則によるトークンユースケース一覧

パーマカルチャーの多機能性原則に基づき、このトークンを「単なる支払い」以外の用途にも使う。

### 確定ユースケース

| ユースケース | トークンの役割 | 実装難易度 |
|-------------|--------------|-----------|
| お手伝いへの報酬 | 労働対価の支払い | 低 |
| 農産物の購入 | 物品交換の媒介 | 低 |
| マッチング依頼の掲示 | 掲示板への投稿デポジット | 低 |

### 検討中ユースケース（多機能性の拡張）

| ユースケース | トークンの役割 | 実装難易度 |
|-------------|--------------|-----------|
| **農機具の共同利用予約権** | 利用権のトークン化（時間単位） | 中 |
| **勉強会・ワークショップの参加チケット** | イベント参加権 | 低 |
| **種の図書館** | 種の貸し借り・返却管理 | 中 |
| **共同農園の区画利用権** | 土地利用権のNFT化 | 高 |
| **地域のお祭り・文化活動への寄付** | コミュニティ財へのDonate | 低 |
| **農業技術・経験のスキル証明** | SBTバッジ発行 | 中 |
| **環境再生活動の証明** | Earth Care SBTとトークン報酬 | 高 |

## 5. Earth Care トークンの設計案

パーマカルチャーの「地球への配慮」倫理を具体的なトークン発行トリガーにする。

```
環境再生活動
  ├─ 堆肥作り（kg単位で換算）
  ├─ 植樹（本数で換算）
  ├─ 農薬不使用の証明（シーズン単位）
  ├─ 水源保全活動（参加時間で換算）
  └─ 生物多様性の記録提出（写真・データ）
         ↓
   Proof of Earth Care
   （活動内容をオラクルまたはコミュニティ承認で検証）
         ↓
   Earth Care SBT の発行（実績バッジ）
         ↓
   ユーティリティトークンの付与
```

> **課題：** 活動の「証明」をどう検証するか。コミュニティによる承認（相互監視）か、写真・GPS記録などのオフチェーンデータとの組み合わせか。

## 6. 設計原則のまとめ

本プロジェクトの設計全体を貫くパーマカルチャー的原則：

| パーマカルチャー原則 | 地域通貨設計への応用 |
|--------------------|-------------------|
| **観察してから介入する** | まず小規模コミュニティで試験運用し、データを見てからスケール |
| **多機能性** | トークンを支払い・予約権・投票権・バッジに多目的活用 |
| **余剰を循環させる** | 使われないトークンは自動的にコミュニティ財へ（Demurrage） |
| **エッジを大切にする** | コミュニティの境界（新参者との接点）を丁寧に設計 |
| **小さく始めてゆっくり成長する** | MVPは30人以下の小規模から、段階的に拡張 |
| **自己調整と自己修正** | DAOガバナンスによる定期的なルールの見直し |
