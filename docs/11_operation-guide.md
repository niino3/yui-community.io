# 運用マニュアル — パーマカルチャー教室向け

**対象**: コミュニティ運営者（operator / admin）  
**バージョン**: v0.2.0（テスト運用版）  
**最終更新**: 2026-03-12

---

## 1. Yui とは

Yui は、パーマカルチャーコミュニティ向けの地域通貨アプリです。

- 農作業を手伝うとトークン（YUI）がもらえる
- トークンで農産物を買ったり、農機具を借りたりできる
- 環境活動（Earth Care）もトークンで評価される
- QRコードで簡単に決済

**現在はテスト版（Polygon Amoy テストネット）です。トークンに現金価値はありません。**

## 3. セットアップ手順

### 3.1 MetaMask をインストール

1. App Store / Google Play で「MetaMask」を検索してインストール
2. 新しいウォレットを作成
3. **シークレットリカバリーフレーズを紙に書いて安全に保管**（これを失うとウォレットを復元できません）

### 3.2 Polygon Amoy テストネットを追加

1. MetaMask を開く
2. ネットワーク選択 → 「ネットワークを追加」
3. 以下を入力:

| 項目 | 値 |
|------|---|
| ネットワーク名 | Polygon Amoy |
| RPC URL | `https://rpc-amoy.polygon.technology` |
| チェーン ID | `80002` |
| 通貨記号 | POL |
| ブロックエクスプローラー | `https://amoy.polygonscan.com` |

### 3.3 テスト用 POL を取得（ガス代）

トークンの送金にはごく少量の POL（ネットワーク手数料）が必要です。

1. https://faucet.polygon.technology/ にアクセス
2. ネットワーク「Polygon Amoy」を選択
3. MetaMask のウォレットアドレスを入力
4. 「Submit」→ 数分で POL が届きます

### 3.4 Yui アプリにアクセス

1. ブラウザで https://yui-community-io.vercel.app/ を開く
2. 「ウォレットを接続」をタップ
3. MetaMask が起動 → 接続を承認
4. ネットワークが Polygon Amoy であることを確認

## 4. 基本操作

### 4.1 トークンを確認する

ホーム画面に以下が表示されます:
- **YUI 残高**: 保有しているトークン数
- **SBT**: メンバーシップ証明（保有 = コミュニティメンバー）

### 4.2 QR コードで支払う

1. ホーム画面の「QR 決済」をタップ
2. 「スキャンして支払う」を選択
3. 相手の QR コードをカメラでスキャン
4. 金額を確認 → 「支払う」
5. MetaMask で署名を承認
6. 「決済完了！」が表示されれば成功

### 4.3 QR コードで受け取る

1. 「QR 決済」→「QR コードで受け取る」
2. 金額を入力（任意）
3. 表示された QR コードを相手に見せる

### 4.4 手動で送金する

1. ホーム画面の「送金」をタップ
2. 送金先ウォレットアドレス（0x...）を入力
3. 金額を入力
4. 「送金する」→ MetaMask で署名

### 4.5 タスクを投稿する

1. 「おたすけ」タブをタップ
2. 「＋ タスクを投稿」をタップ
3. タイトル・説明・カテゴリ・報酬（YUI）を入力
4. 「投稿する」

### 4.6 タスクに応募する

1. タスク一覧から気になるタスクをタップ
2. 内容を確認 →「応募する」

### 4.7 作業完了→承認の流れ

1. **作業者**: 作業完了後、タスク詳細画面で「作業完了報告」をタップ
2. **依頼者**: 完了報告を確認 →「承認する」をタップ
3. トークンが自動的に作業者に送金されます

## 5. 運営者向け操作

### 5.1 メンバーにトークンを配布する

テスト運用では、運営者がメンバーに初期トークンを配布します。

1. MetaMask を開く
2. YUI トークンを「送信」
3. メンバーのウォレットアドレスを入力
4. 配布数量を入力して送信

**推奨初期配布量**: 100 YUI / 1人

### 5.2 コミュニティ統計を確認する

1. プロフィール画面 →「コミュニティ統計」
2. 以下が確認できます:
   - 総メンバー数
   - 今月のアクティブユーザー数
   - 総取引件数
   - 流通トークン量

### 5.3 SBT（メンバーシップ証明）を発行する

新しいメンバーが参加したら、SBT を発行してコミュニティメンバーであることを証明します。

> 現在はコントラクトのオーナーのみが発行可能です。管理画面からの発行機能は今後追加予定です。

## 7. よくある質問 (FAQ)

### Q. トークンは現金に換えられますか？

**いいえ。** YUI トークンはコミュニティ内でのみ使える「お手伝いの証」です。法定通貨への交換は設計上できません。

### Q. MetaMask のパスワードを忘れました

シークレットリカバリーフレーズがあれば復元できます。フレーズも失った場合、残念ながらウォレットは復元できません。新しいウォレットを作成し、運営者に連絡してください。

### Q. 送金できません（エラーが出ます）

以下を確認してください:
1. **ネットワーク**: Polygon Amoy になっているか
2. **POL 残高**: ガス代用の POL があるか（Faucet で無料取得可能）
3. **YUI 残高**: 送金に十分なトークンがあるか

### Q. QR コードが読み取れません

- カメラの権限を許可しているか確認
- 明るい場所で試す
- QR コードが画面に大きく表示されるようにする

### Q. テストネットと本番の違いは？

| | テストネット（Amoy） | 本番（Polygon PoS） |
|---|---|---|
| トークンの価値 | なし（テスト用） | コミュニティ内で有効 |
| ガス代 | 無料（Faucet） | ごく少額の POL が必要 |
| 用途 | 動作検証・練習 | 実運用 |

## 9. ローカル開発環境のセットアップ

### 9.1 前提条件

| ソフトウェア | 用途 |
|-------------|------|
| Node.js 18+ | フロントエンド開発 |
| Docker Desktop | バックエンド（Laravel + PostgreSQL + Redis） |
| Git | バージョン管理 |

### 9.2 バックエンド起動

```bash
cd backend
docker-compose up
```

4つのコンテナが起動します:
- `yui-app` — PHP-FPM（Laravel）
- `yui-nginx` — Nginx（ポート 8080）
- `yui-db` — PostgreSQL
- `yui-redis` — Redis

初回のみマイグレーションを実行:
```bash
docker-compose exec app php artisan migrate
```

### 9.3 フロントエンド起動

```bash
# プロジェクトルートで
npm install
npm run dev
```

`.env` ファイルでバックエンド接続先を設定:
```
VITE_API_URL=http://localhost:8080
```

### 9.4 3つのサイトへのアクセス

| サイト | URL | アクター |
|--------|-----|---------|
| Site 1: Platform Admin | `http://localhost:5173/` | Service Owner |
| Site 2: Community Admin | `http://localhost:5173/?community={slug}&admin=true` | Community Owner |
| Site 3: Community App | `http://localhost:5173/?community={slug}` | Community Member |

例: `test-farm` コミュニティの場合:
- 管理画面: `http://localhost:5173/?community=test-farm&admin=true`
- アプリ: `http://localhost:5173/?community=test-farm`

### 9.5 CORS 設定

フロントエンドのポートが変わった場合（例: 5174）、`backend/config/cors.php` の `allowed_origins` に追加してください:

```bash
docker-compose exec app php artisan config:clear
```
