## P1：田中 よし子（72歳）― 農家・依頼者

### プロフィール

```
職業    : 専業農家（野菜・果物）。夫と2人暮らし。
背景    : 40年以上この地で農業を続けてきたが、
          高齢になり繁忙期の草取り・収穫作業が体力的に厳しくなってきた。
動機    : 「お金を払って雇う」のは気が引ける。
          昔みたいに「お互い様」で助け合えるコミュニティを取り戻したい。
技術    : スマホはLINEと電話のみ。QRコードはコンビニで経験あり。
          ウォレット・ブロックチェーンは全く知らない。
不安    : 「難しい操作は無理」「個人情報が漏れないか」「騙されないか」
```

### ユーザーストーリー

| # | ストーリー | 受け入れ条件 |
|---|-----------|------------|
| P1-1 | **草取りの手伝いを依頼したい**<br>畑の草取りを誰かに頼めるよう、依頼を掲示板に投稿したい | 農作業の種類・日時・場所・報酬（トークン量）を入力して投稿できる |
| P1-2 | **応募者を確認して選びたい**<br>応募してくれた人の顔・過去の実績を見てから承認したい | 応募者の信頼スコア・過去の取引件数・評価コメントを確認できる |
| P1-3 | **作業完了を簡単に承認したい**<br>スマホ操作が苦手でも、作業が終わったことを確認・承認できる | QRコードをかざすだけで承認が完了し、トークンが自動送金される |
| P1-4 | **自分の農産物をトークンで売りたい**<br>直売所で野菜にQRタグをつけて、トークンで購入してもらいたい | QRタグを印刷して農産物に貼り付け、購入者がスキャンして決済できる |
| P1-5 | **残高を確認したい**<br>自分のトークンがどれくらいあるか知りたい | LINEのような画面でトークン残高と取引履歴が確認できる |

## P2：鈴木 けんた（34歳）― 移住者・週末農業・作業者

### プロフィール

```
職業    : 平日はリモートワーク（Web系）。2年前に都市部から移住。
背景    : 農業に興味があり、週末に近所の農家を手伝いたいが
          知り合いが少なく、どこに声をかければいいかわからない。
動機    : コミュニティに貢献しながらトークンを貯め、
          農産物や農機具を借りるために使いたい。
          ゆくゆくは自分でも農地を借りて農業をしたい。
技術    : スマホは普通に使える。Web3は初心者だが学習意欲はある。
不安    : 「どの農家に声をかけていいかわからない」
          「初対面の人との取引が不安」
```

### ユーザーストーリー

| # | ストーリー | 受け入れ条件 |
|---|-----------|------------|
| P2-1 | **今週末に手伝える農作業を探したい**<br>日程・場所・種類でフィルタリングして依頼を探せる | 地図表示・日程・作業種別でフィルタリングできる一覧画面がある |

| P2-6 | **自分の貢献実績を見たい**<br>積み重ねたSBTや信頼スコアを確認・自己紹介に使いたい | 貢献SBT・Earth Care SBT・信頼スコア・取引履歴のプロフィール画面がある |

## P3：山田 さやか（28歳）― コミュニティ運営者

### プロフィール

```
職業    : 地域おこし協力隊。コミュニティの活性化を担当。
背景    : Web・SNS運営が得意。このプロジェクトの立ち上げを推進。
          コミュニティメンバーのオンボーディングやサポートが主な役割。
動機    : 地域の「見えない貢献」を可視化して、
          若者や移住者が活躍できる仕組みを作りたい。
技術    : スマホ・PC両方使える。Web3の基礎知識あり。
不安    : 「高齢者がついてこられるか」「参加者が増えないと意味がない」
```

### ユーザーストーリー

| # | ストーリー | 受け入れ条件 |
|---|-----------|------------|
| P3-1 | **新メンバーをオンボーディングしたい**<br>スマホが苦手な高齢農家でも3分以内にアカウント開設できるよう案内したい | LINEログイン → メンバーシップSBT発行 → チュートリアルが10タップ以内で完了する |

## P4：佐藤 ひろし（45歳）― 地域住民・非農家

### プロフィール

```
職業    : 会社員（製造業）。週末が休み。
背景    : 農業経験はないが、地域に貢献したい気持ちがある。
          農機具（耕運機）を借りて家庭菜園を始めたい。
動機    : トークンを稼いで農機具を借りたい。
          地域の人とつながるきっかけが欲しい。
技術    : スマホは普通に使える。QRコード・PayPayの経験あり。
不安    : 「農作業の経験がなくて役に立てるか」「農機具の使い方がわからない」
```

### ユーザーストーリー

| # | ストーリー | 受け入れ条件 |
|---|-----------|------------|
| P4-1 | **農業未経験でもできるタスクを探したい**<br>草取り・収穫の袋詰めなど、特別なスキルが不要な作業に絞って探せる | スキルレベル（未経験OK / 経験者向け）でフィルタリングできる |
| P4-2 | **農機具を借りたい**<br>耕運機を使いたい日時で予約し、トークンで利用料を払いたい | 農機具一覧から選んで日時を選択し、トークンでエスクロー予約ができる |
| P4-3 | **農機具の使い方を確認したい**<br>農機具QRを読んだら操作マニュアルと注意事項が表示されてほしい | QRスキャン後に動画マニュアル・緊急連絡先・保険情報が表示される |
| P4-4 | **農機具を返却してトークンを精算したい**<br>返却時のQRスキャンで利用料が自動計算されて精算されてほしい | 返却QRスキャン → 利用時間・料金確認 → 精算完了が3タップで終わる |
